01 — Vấn đề

Chạy GoClaw cần cả một hệ sinh thái

GoClaw Server yêu cầu PostgreSQL, Redis (optional), và Docker để deploy. Đây là điều hoàn toàn hợp lý với production multi-tenant — nhưng với developer muốn thử nghiệm nhanh hoặc dùng cá nhân, đó là quá nhiều friction.

GoClaw Lite giải quyết điều đó. Một file .app duy nhất. Click để chạy. SQLite thay PostgreSQL. Wails thay Docker.

GoClaw Lite dành cho single-user local deployment. Không có multi-tenant. Không expose trên LAN (bind 127.0.0.1). Đây là đánh đổi có chủ đích.

03 — Kiến trúc

Một process, hai thế giới

04 — Wails IPC

Go backend, React frontend — không qua mạng

Wails v2 inject Go methods vào window.go của WebView. Frontend gọi trực tiếp như gọi JavaScript function — không HTTP, không fetch, không port. Latency gần bằng zero.

TypeScript — wails.ts 
// Type-safe Wails bindings — no fetch, no HTTP
export const wails = {
  getGatewayURL: (): Promise<string>  => window.go.main.App.GetGatewayURL(),
  getGatewayToken: (): Promise<string>  => window.go.main.App.GetGatewayToken(),
  isGatewayReady: (): Promise<boolean> => window.go.main.App.IsGatewayReady(),
  checkForUpdate: (): Promise<UpdateInfo> => window.go.main.App.CheckForUpdate(),
  applyUpdate: (): Promise<void> => window.go.main.App.ApplyUpdate(),
  resetDatabase: (): Promise<void> => window.go.main.App.ResetDatabase(),
}

Security note: ApplyUpdate() không nhận URL từ frontend — chỉ dùng UpdateInfo đã được cache server-side. Frontend không thể trỏ update về URL tùy ý.

05 — SQLite

Pure-Go SQLite — không CGo

GoClaw Lite dùng modernc.org/sqlite — pure Go implementation, không CGo, không native binaries. Cross-compile từ macOS sang Windows trong một lệnh duy nhất.

WAL mode được bật mặc định trên mọi connection. Concurrency được xử lý qua busy_timeout thay vì immediate SQLITE_BUSY error.

journal_mode
= WAL

Write-Ahead Logging cho phép đọc đồng thời trong khi ghi. Không lock toàn bộ database.

busy_timeout
= 15000

Chờ tối đa 15 giây khi DB bị lock. Tránh SQLITE_BUSY lỗi ngay lập tức.

synchronous
= NORMAL

Cân bằng giữa an toàn và performance. Không fsync mỗi transaction.

cache_size
= -8000

8MB page cache trong bộ nhớ. Âm = kilobytes, dương = số trang.

Go — sqlitestore/pool.go 
// pragmaConnector wraps sql.Driver để apply PRAGMAs trên MỌI connection.
// db.Exec() chỉ apply cho 1 connection — các connection khác sẽ bị SQLITE_BUSY.
var connectionPragmas = []string{
    "PRAGMA journal_mode = WAL",
    "PRAGMA busy_timeout = 15000",  // 15s wait
    "PRAGMA synchronous = NORMAL",
    "PRAGMA cache_size = -8000",      // 8MB
    "PRAGMA foreign_keys = ON",
}

// Build tag: //go:build sqlite || sqliteonly
// Compile: go build -tags sqliteonly .
06 — Auto-Update

Tự động cập nhật từ GitHub Releases

Không cần mở browser. Không cần tìm download link. GoClaw Lite kiểm tra GitHub Releases mỗi 6 giờ và thông báo khi có bản mới. Một click để update và restart.

STEP 01

Startup + 5s delay

Sau khi gateway sẵn sàng, check update lần đầu. Sau đó mỗi 6 giờ.

STEP 02

Query GitHub API

Tìm release với tag lite-v*. So sánh semver với version hiện tại.

STEP 03

Download + Verify

Download asset. URL được cache server-side — frontend không thể inject URL tùy ý.

STEP 04

Apply + Restart

WAL checkpoint, replace binary, relaunch. SQLite data nguyên vẹn.

Go — app.go 
// updateLoop: check on startup (5s delay) then every 6h
func (a *App) updateLoop() {
    timer := time.NewTimer(5 * time.Second)
    select {
    case <-timer.C:
        a.checkAndEmitUpdate()
    case <-a.ctx.Done():
        return
    }
    ticker := time.NewTicker(6 * time.Hour)
    for {
        select {
        case <-ticker.C:
            a.checkAndEmitUpdate()
        case <-a.ctx.Done():
            return
        }
    }
}

08 — Cài đặt

Cài đặt trong 30 giây

One-liner cho macOS. Windows dùng PowerShell hoặc download thủ công từ GitHub Releases.

curl -fsSL https://raw.githubusercontent.com/nextlevelbuilder/goclaw/main/scripts/install-lite.sh | bash

Cài vào /Applications/goclaw-lite.app. Tự động xóa quarantine attribute.

iwr -useb https://raw.githubusercontent.com/nextlevelbuilder/goclaw/main/scripts/install-lite.ps1 | iex

PowerShell. Download .zip và extract vào %LOCALAPPDATA%\GoClaw Lite.

https://github.com/nextlevelbuilder/goclaw/releases → Tìm tag lite-v* → Download: goclaw-lite-{version}-darwin-arm64.tar.gz

Hỗ trợ: macOS (arm64, amd64), Windows (amd64). Linux chưa được hỗ trợ.

Data location: ~/.goclaw/data/goclaw.db — SQLite database, WAL files, và media storage. Backup bằng cách copy thư mục này. Secrets (encryption key, gateway token) được lưu trong OS keyring.
